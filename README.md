# Employee Database Analysis using SQL

## Tools Used
- SQL Server / MySQL

## Concepts Used
- Joins
- Group By
- Aggregate Functions
- Subqueries
- Window Functions

## Business Questions Solved
1. List all employees.
2. Find the highest-paid employee.
3. Find average salary by department.
4. Rank employees by salary.
5. Find employees hired after 2023.
6. Count employees in each department.
7. Find employees with salary above average.
8. Find the second-highest salary.
9. Find the department with the highest average salary.
10. Show employee details with department and salary.
